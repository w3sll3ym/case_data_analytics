
DOCUMENTAÇÃO DO SCRIPT PYTHON (index.py)

Este script converte arquivos JSON corrigidos em arquivos CSV, para facilitar a análise e o uso em plataformas SQL.

1. Bibliotecas Utilizadas:
   - json: Para ler os arquivos JSON.
   - csv: Para criar e escrever arquivos CSV.

2. Estrutura do Código:
   - O script lê dois arquivos JSON (fixed_database_1.json e fixed_database_2.json).
   - Para cada arquivo, ele:
     1. Lê o conteúdo do JSON e carrega os dados em uma variável.
     2. Cria um arquivo CSV correspondente (fixed_database_1.csv e fixed_database_2.csv).
     3. Escreve as colunas e os dados no CSV.

3. Uso:
   - Certifique-se de que os arquivos JSON corrigidos estão no mesmo diretório do script.
   - Execute o script em um ambiente Python configurado.
   - Os arquivos CSV serão gerados no mesmo diretório.

4. Pontos de Atenção:
   - Verifique se os arquivos JSON têm o formato esperado (listas de dicionários).
   - O script sobrescreve os arquivos CSV existentes com o mesmo nome.
